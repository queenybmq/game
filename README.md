# 星舰幸存者

一款无依赖的 HTML5 Canvas 科幻幸存者游戏。玩家驾驶最后一艘星舰，在霓虹星域中自动射击、拾取经验、选择升级、同步主动技能，并在不断增强的敌潮里挑战虚空母舰。

## 项目特点

- **纯前端实现**：无需后端服务，核心由 `index.html`、`styles.css`、`game.js` 组成。
- **Canvas 实时战斗**：包含敌群生成、自动瞄准、投射物、拾取物、粒子、屏幕震动与动态 HUD。
- **Roguelite 升级系统**：升级时从 3 个候选强化中选择 1 个，构建不同武器、防御和机动路线。
- **多主武器与主动技能**：包含脉冲炮、飞弹、轨道长钉炮、链式电弧、绿焰霰弹，以及虚空长矛、重力井、相位闪烁、离子风暴等主动技能。
- **随机事件与星域轮换**：战斗中会触发补给、经验彗星、精英信号、磁场脉冲、虚空乱流等事件。
- **Boss 与超载波次**：虚空母舰登场后进入高压战斗，击破后会启动超载波次。
- **多语言支持**：支持 English、中文简体、中文繁體、日本語，默认语言为中文简体。
- **本地记录保存**：最高分、最长生存时间、最高击杀和最高等级会保存到浏览器本地。
- **性能自适应**：提供画质模式、自动降低特效压力、动态效果开关，敌人和特效过多时会自动降级部分绘制成本。

## 快速开始

直接用浏览器打开 `index.html` 即可运行。也可以在项目根目录启动本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/
```

## 操作说明

| 操作 | 说明 |
| --- | --- |
| 鼠标移动 | 星舰跟随鼠标指针移动 |
| 自动射击 | 星舰会自动瞄准并攻击附近敌人 |
| 鼠标左键 | 释放已同步的主动技能 |
| `P` / `Esc` | 暂停或恢复游戏 |
| `O` | 打开游戏设置 |
| 菜单右上角语言选择 | 切换界面语言 |
| 返回主菜单 | 保留当前战局，并可从菜单继续 |

## 游戏流程

1. 在主菜单点击 **开始新游戏**。
2. 控制星舰躲避敌人和弹幕。
3. 击败敌人获得经验晶体。
4. 升级后选择一项强化。
5. 同步主动技能或强化主武器构建战斗路线。
6. 挑战精英舰和虚空母舰。
7. 游戏结束后查看本局表现和历史最佳纪录。

## 主动技能

开局升级时会从主动技能池中选择一个，本局只能同步一个主动技能，之后可以继续升级到 4 级。

| 技能 | 玩法 |
| --- | --- |
| 虚空长矛 | 向鼠标方向发射穿透长矛，对路径上的敌人造成高额伤害。 |
| 重力井 | 在鼠标位置生成牵引力场，拉扯并持续伤害范围内敌人。 |
| 相位闪烁 | 向鼠标方向短距离闪现，并在起点和落点释放冲击波。 |
| 离子风暴 | 呼叫多道离子雷击，优先打击附近敌人。 |

## 升级技能

升级池包含攻击、防御、机动、特殊、Build 和主武器强化：

- 攻击：脉冲炮强化、分裂雷射、等离子飞弹、星核爆裂、暴击校准。
- 防御：护盾扩容、奈米修复、相位闪避。
- 机动：引擎超频、磁力回收。
- 特殊：环绕无人机、电磁脉冲。
- Build：虹吸护盾、拾取爆裂、近距离歼灭、临界火力、主动电池、不稳定弹药。
- 主武器：轨道长钉炮、链式电弧、绿焰霰弹。

这些强化可以叠加出不同流派，例如飞弹追踪、暴击爆裂、低血量临界火力、护盾虹吸、拾取爆裂或多武器并行输出。

## 核心玩法

游戏目标是在敌潮中尽可能久地生存并打出高分。敌人死亡后掉落经验晶体，拾取经验会提升等级并触发升级选择。分数来自生存时间、击杀、经验拾取和 Boss 击破奖励。

战场会随时间推进：敌人数量上限、刷新速度、生命值和移动速度都会逐步提高。第 34 秒后会出现精英舰，约第 240 秒虚空母舰登场。击破虚空母舰后会进入超载波次，敌潮压力和分数增长进一步提升。

## 敌人、星域与事件

敌人包括追猎机、装甲舰、散射机、自爆蜂群、星云幽梭、碎岩钻机、雷暴导体、遗迹哨兵、精英舰和虚空母舰。普通敌人还可能出现富矿或迅捷变体。

星域会在星云残域、碎岩环带、离子风暴、机械遗迹之间切换。不同星域会提高对应特色敌人的出现概率。

随机事件会周期性触发，包括经验彗星、精英信号、补给舱、晶体绽放、不稳定核心、磁场脉冲、超频窗口和虚空乱流。

## 拾取物与临时增益

拾取物包括经验、修复、护盾、爆裂核心、磁力和超频。磁力会短时间扩大回收范围，超频会短时间提高伤害和射速，爆裂核心会立即造成范围伤害。

## 设置项

游戏设置中包含：

- 音效音量
- 画质模式：高画质、平衡、性能优先
- 画面震动
- 动态扫描效果
- 自动降低特效压力

语言选择已独立到主菜单右上角，方便进入游戏前快速切换。

## 多语言

当前支持语言：

- English
- 中文简体
- 中文繁體
- 日本語

语言配置集中在 `game.js` 的国际化数据中。切换语言后，菜单、HUD、升级卡、提示、事件、Boss 文本等动态内容都会同步更新。

## 性能说明

游戏针对高压战斗做了多项优化：

- 根据敌人、子弹和粒子数量动态判断性能压力。
- 高压时自动减少粒子、浮字、阴影光晕和部分背景效果。
- 子弹碰撞先进行快速范围排除，减少不必要计算。
- 敌人分离逻辑在高压时降低检查频率。
- 可通过设置中的画质模式进一步控制性能表现。

如果设备性能较弱，建议使用 **性能优先** 模式，并开启 **自动降低特效压力**。

## 本地数据

游戏使用 `localStorage` 保存：

- 游戏设置
- 最高分
- 最长生存时间
- 最高击杀
- 最高等级

清除浏览器站点数据会重置这些记录。
