# 星舰幸存者

一个纯前端 HTML5 Canvas 科幻幸存者游戏。玩家驾驶最后一艘星舰，在霓虹星海中自动射击、拾取经验、选择升级、同步主动技能，并挑战不断增强的虚空母舰威胁。

## 项目特点

- **纯前端实现**：无需后端服务，核心由 `index.html`、`styles.css`、`game.js` 组成。
- **Canvas 实时战斗**：包含敌群生成、自动瞄准、投射物、拾取物、粒子、屏幕震动与动态 HUD。
- **Roguelite 升级系统**：升级时从多张强化卡中选择，构建不同武器与防御路线。
- **多主武器与主动技能**：包含脉冲炮、飞弹、轨道长钉、链式电弧、霰弹，以及虚空长矛、重力井、相位闪烁、离子风暴等主动技能。
- **随机事件与星域轮换**：战斗中会触发补给、经验彗星、精英信号、磁场脉冲、虚空乱流等事件。
- **Boss 与超载波次**：虚空母舰登场后进入高压战斗，击破后会启动超载波次。
- **多语言支持**：支持 English、中文简体、中文繁體、日本語，默认语言为中文简体。
- **本地记录保存**：保存最高分、最长生存时间、最高击杀等纪录。
- **性能自适应**：提供画质模式、自动降低特效压力、动态效果开关，敌人和特效过多时会自动降级部分绘制成本。

## 快速开始

### 方式一：直接打开

用浏览器打开 `index.html` 即可运行。

### 方式二：启动本地静态服务

如果浏览器或环境对本地文件访问有限制，可以在项目目录启动一个静态服务：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

也可以使用任意静态服务器，例如 VS Code Live Server、`npx serve` 等。

## 操作说明

| 操作 | 说明 |
| --- | --- |
| 鼠标移动 | 星舰跟随鼠标指针移动 |
| 自动射击 | 星舰会自动瞄准并攻击附近敌人 |
| 鼠标左键 | 释放已同步的主动技能 |
| P / Esc | 暂停或恢复游戏 |

## 游戏流程

1. 在主菜单点击 **开始新游戏**。
2. 控制星舰躲避敌人和弹幕。
3. 击败敌人获得经验晶体。
4. 升级后选择一项强化。
5. 同步主动技能或强化主武器构建战斗路线。
6. 挑战精英舰和虚空母舰。
7. 游戏结束后查看本局表现和历史最佳纪录。

## 设置项

游戏设置中包含：

- 音效音量
- 画质模式
  - 高画质
  - 平衡
  - 性能优先
- 画面震动
- 动态扫描效果
- 自动降低特效压力

语言选择已独立到主菜单右上角，方便进入游戏前快速切换。

## 多语言

当前支持语言：

- English
- 中文简体
- 中文繁體
- 日本語

语言配置集中在 `game.js` 的国际化数据中。切换语言后，菜单、HUD、升级卡、提示、事件、Boss 文本等动态内容都会同步更新。

## 性能说明

游戏针对高压战斗做了多项优化：

- 根据敌人、子弹和粒子数量动态判断性能压力。
- 高压时自动减少粒子、浮字、阴影光晕和部分背景效果。
- 子弹碰撞先进行快速范围排除，减少不必要计算。
- 敌人分离逻辑在高压时降低检查频率。
- 可通过设置中的画质模式进一步控制性能表现。

如果设备性能较弱，建议使用 **性能优先** 模式，并开启 **自动降低特效压力**。

## 本地数据

游戏使用 `localStorage` 保存：

- 游戏设置
- 最高分
- 最长生存时间
- 最高击杀
- 最高等级

清除浏览器站点数据会重置这些记录。

## 开发说明

本项目不依赖构建工具，修改后刷新浏览器即可查看效果。

建议修改后至少执行：

```bash
node --check game.js
```

该命令可以检查 JavaScript 语法是否有效。

## 浏览器兼容

建议使用较新的桌面浏览器：

- Chrome / Edge
- Firefox
- Safari

游戏使用 Canvas、Web Audio、LocalStorage、现代 CSS 与 JavaScript 特性，旧版浏览器可能无法完整支持。
