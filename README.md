# 星舰幸存者

一款无依赖的浏览器 Canvas 生存射击游戏。玩家驾驶星舰在霓虹星域中自动射击、拾取经验、选择升级，并在不断增强的敌潮里挑战虚空母舰。

## 运行方式

直接用浏览器打开 `index.html` 即可游玩。也可以在项目根目录启动本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://127.0.0.1:8000/`。

## 操作方式

- 鼠标指针牵引星舰移动。
- 武器会自动瞄准并自动射击最近敌人。
- 左键释放当前主动技能。
- `P` / `Esc` 暂停或继续游戏。
- `O` 打开游戏设置。
- 返回主菜单可以保留当前战局，并从菜单继续。

## 内容数量

当前版本包含：

- 主动技能：4 个。
- 升级技能：21 个。
- 可成长技能项合计：25 个。
- 敌人类型：10 种。
- 拾取物：6 种。
- 星域地图：4 个。
- 随机事件：8 类。
- 敌潮阶段：6 段。

## 主动技能

开局升级时会从主动技能池中选择一个，本局只能同步一个主动技能，之后可以继续升级到 4 级。

| 技能 | 玩法 |
| --- | --- |
| 虚空长矛 | 向鼠标方向发射穿透长矛，对路径上的敌人造成高额伤害。 |
| 重力井 | 在鼠标位置生成牵引力场，拉扯并持续伤害范围内敌人。 |
| 相位闪烁 | 向鼠标方向短距离闪现，并在起点和落点释放冲击波。 |
| 离子风暴 | 呼叫多道离子雷击，优先打击附近敌人。 |

## 升级技能

升级时从 3 个候选强化中选择 1 个。升级池包含攻击、防御、机动、特殊、Build 和主武器强化：

- 攻击：脉冲炮强化、分裂雷射、等离子飞弹、星核爆裂、暴击校准。
- 防御：护盾扩容、奈米修复、相位闪避。
- 机动：引擎超频、磁力回收。
- 特殊：环绕无人机、电磁脉冲。
- Build：虹吸护盾、拾取爆裂、近距离歼灭、临界火力、主动电池、不稳定弹药。
- 主武器：轨道长钉炮、链式电弧、绿焰霰弹。

这些强化可以叠加出不同流派，例如飞弹追踪、暴击爆裂、低血量临界火力、护盾虹吸、拾取爆裂或多武器并行输出。

## 核心玩法

游戏目标是在敌潮中尽可能久地生存并打出高分。敌人死亡后掉落经验晶体，拾取经验会提升等级并触发升级选择。分数来自生存时间、击杀、经验拾取和 Boss 击破奖励。

战场会随时间推进：敌人数量上限、刷新速度、生命值和移动速度都会逐步提高。第 34 秒后会出现菁英舰，约第 240 秒虚空母舰登场。击破虚空母舰后会进入超载波次，敌潮压力和分数增长进一步提升。

## 敌人、星域与事件

敌人包括追猎机、装甲舰、散射机、自爆蜂群、星云幽梭、碎岩钻机、雷暴导体、遗迹哨兵、菁英舰和虚空母舰。普通敌人还可能出现富矿或迅捷变体。

星域会在星云残域、碎岩环带、离子风暴、机械遗迹之间切换。不同星域会提高对应特色敌人的出现概率。

随机事件会周期性触发，包括经验彗星、菁英讯号、补给舱、晶体绽放、不稳定核心、磁场脉冲、超频窗口和虚空乱流。

## 拾取物与临时增益

拾取物包括经验、修复、护盾、爆裂核心、磁力和超频。磁力会短时间扩大回收范围，超频会短时间提高伤害和射速，爆裂核心会立即造成范围伤害。

## 设置与记录

游戏设置支持音量、画质、画面震动、动态扫描效果和自动降低特效压力。最高分、最长生存时间、最高击杀和最高等级会保存在浏览器 `localStorage` 中。

## 项目结构

- `index.html`：页面结构、菜单、HUD、升级面板和设置面板。
- `styles.css`：视觉主题、响应式布局、HUD 和动效。
- `game.js`：游戏循环、输入、战斗、敌潮、技能、升级、存档和渲染逻辑。
